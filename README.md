# Matlab_Microgrid

You must design a new microgrid system that will provide uninterrupted power over a seven sol period (1 sol = 24 hrs 37 minutes). The system must also keep voltage between 500V and 520V at all times, as various sensitive pieces of life-support equipment cannot tolerate voltage levels outside these bounds
